Welcome to Llama List!


Meet our new integration for Microsoft Teams, Outlook, and Office.


Once you have added this app, simply navigate to the Llama List tab where you will be prompted to sign in once. From there you can create and edit tasks in your Llama List!


If you are using Microsoft Teams, you can turn any message/chat into a task. You can do so by navigating to the "more actions" button on the message bubble and clicking "Create Task in Llama List".


You must be a Llama List user in order to use this add-in. To create an account sign up here
